Authenticate a subuser on your website before displaying their account information so that you can have users manage their SendGrid account on your website.

Parameter Required Requirements
user Yes Subuser that is registered under your account.
password Yes Password the subuser submitted.
{% apiexample auth POST https://api.sendgrid.com/apiv2/customer.auth api_user=your_sendgrid_username&api_key=your_sendgrid_password&user=example@example.com&password=theirsubmittedpassword %} {% response json %} { "message": "success" } //error { "message": "error", "errors": [ "...error messages..." ] } {% endresponse %} {% response xml %} success error ... ... error messages ... ... {% endresponse %} {% endapiexample %}
